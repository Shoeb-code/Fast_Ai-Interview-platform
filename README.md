


# 🚀 AI Personal Interviewer Platform

![React](https://img.shields.io/badge/React-Frontend-blue)
![Node](https://img.shields.io/badge/Node-Backend-green)
![MongoDB](https://img.shields.io/badge/MongoDB-Database-darkgreen)
![AI](https://img.shields.io/badge/Gemini-AI-purple)

A **premium AI-powered mock interview SaaS platform** built with the **MERN Stack** that helps candidates prepare for interviews through intelligent AI-generated questions, answer evaluation, real-time scoring, analytics, and personalized feedback.

Designed with a **modern SaaS-grade UI**, OTP authentication, profile analytics, and performance tracking.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ai-personal-interviewer.git
```

---

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

### Backend Setup

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create `.env` file inside backend:

```env
PORT=5000

MONGO_URI=your_mongodb_url

JWT_SECRET=your_jwt_secret

EMAIL_USER=your_email
EMAIL_PASS=your_email_app_password

GEMINI_API_KEY=your_api_key
```
